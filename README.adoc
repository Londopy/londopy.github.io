:toc: macro

Curated showcase of https://github.com/Londopy[@Londopy]'s open-source projects, grouped by domain and filterable by tag or language. Built with https://astro.build[Astro].

🌐 Live at https://londopy.github.io[*londopy.github.io*]

toc::[]

== Develop

[source,sh]
----
npm install
npm run dev      # local dev server
npm run build    # static build → dist/
----

== Add or edit a project

Everything on the grid comes from `src/data/projects.json` — no component changes needed:

* `projects[]` — one entry per card. Fields: `cluster` (id from `clusters[]`), `name`, `tagline`, `language` (use `"Rust + Python"` for multiple language dots), `tags[]`, `repo`, `demo?`, `page?`, `featured?`, `stars?`. Buttons are omitted when a link is `null`; a `pypi.org` demo automatically becomes a copyable `pip install` line.
* `clusters[]` — section order, emoji, title, and blurb. Empty clusters are hidden automatically.
* `inProgress[]` — the 🚧 teaser strip for reserved repo names.

Set `"featured": true` to also pin a card to the top section. On the homepage, every tag and language dot is a clickable filter.

== Per-project pages

Long-form writeups live in `src/pages/projects/*.astro` and share the `ProjectDoc` layout. Point a project's `page` field at `/projects/<name>/` to surface a `docs →` button on its card.

== Structure

[source]
----
src/
  data/projects.json   # source of truth for every card
  components/          # Hero, Nav, ProjectCard, ClusterSection,
                       # InProgress, BinaryIntro, Footer
  layouts/             # Base, ProjectDoc
  pages/               # index, about, 404, projects/*
  styles/global.css
public/                # favicon.svg, og-image.png, robots.txt
scripts/make_og.py     # regenerates the OG image
----

== Deploy

Pushes to `main` deploy via GitHub Actions (`.github/workflows/deploy.yml`); a `concurrency` group serializes deployments so overlapping runs queue instead of colliding.

One-time setup: *Settings → Pages → Source: GitHub Actions*.

== Regenerate the OG image

`scripts/make_og.py` (requires Pillow + the DejaVu fonts, run from the repo root):

[source,sh]
----
python3 scripts/make_og.py
----
